Title of module: Web Development

Credit Value: 3

Prerequisite: Required

Course Description: This course provides an overview of Internet application development. It introduces the fundamentals of Internet communication and protocols along with the basic knowledge of web programming and design. It introduces the main concepts of client-side and server-side programming. This course is the basic step for students to enrich their knowledge and skills in web development and pursue with more advanced topics in this field.

Aims:

  • Study Internet programming and Web application development.
  • Learn basic principles and techniques for building Internet applications.
  • Introducing client-side and server-side programming.
  • By completion of this course, students will be able to develop a fully functional web application.

Textbooks used:

  • Deitel P. J, Deitel H. M., Internet & World Wide Web: How to Program, (2012), 5th Edition, Prentice Hall. ISBN: 9780132151009
  • Paul McFedries, Web Coding & Development: All-in-One, (2018), John Wiley & Sons. ISBN: 9781119473794
